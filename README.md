 # Resumo do Projeto:
 O projeto eu basiei pela conta de privacidade do Instagram, ele tem os menus: Privacidade, Status da conta, Conta, Ajuda e Sair.
- Privacidade: esta a parte visual somente com as opções de configurar.
- Conta: está com a simulação da API TanStack Query com Nome e email, quando você clica em salvar e faz a "requisição" e da a mensagem que salvou o cadastro.
- Ajuda: Tela somente visual mesmo.
- Sair: quando clicado ele abre um DynamicDialog com a seguinte mensagem 'Tem certeza que deseja sair?', com as opções de confirmar e cancelar.

O projeto segue com o responsivo, colocando o menu na parte superior e exibindo somente o menu quando necessario.
Na parte do menu tem um botão com icone de sol que quando clicado ele muda para lua e coloca toda a tela em modo escuro.
